# 개취 설문조사 프로그램 설계 문서

## 1. 프로젝트 개요

사용자의 개인 취향을 조사하고 결과를 데이터베이스에 저장하여 집계 결과를 확인할 수 있는 설문조사 프로그램을 개발한다.

개발 언어는 Python을 사용하며, MySQL 데이터베이스와 연동한다.

---

## 2. 요구사항

### 입력 항목

1. 생년월 (YYYYMM)
2. 성별
3. 오방색 선호도
4. 평면모양 선호도
5. 입체모양 선호도

### 기능

* 설문 참여
* 설문 결과 저장
* 설문 결과 집계
* 백분율 통계 제공

---

## 3. 시스템 구성

### Main_Survey.py

프로그램 실행 및 메뉴 제어

기능

* 설문 참여
* 통계 조회
* 프로그램 종료

### Func_Info.py

사용자 입력 처리

기능

* 생년월 검증
* 선택지 입력 검증

### Union_DB.py

데이터베이스 처리

기능

* DB 연결
* 테이블 생성
* 설문 저장
* 통계 조회

---

## 4. 데이터베이스 설계

테이블명 : Survey_INFO

| 컬럼명		| 타입			| 설명    	|
| ----------		| ---------- 		| ----- 	|
| NO      		| INT        		| 기본키  	|
| BIRTH    	  	| VARCHAR(6)	| 생년월  	|
| SEX       		| TINYINT   		| 성별    	|
| FIVECOLOR	| TINYINT    	| 오방색  	|
| RP         		| TINYINT    	| 평면모양  	|
| PS         		| TINYINT    	| 입체모양  	|
| CREATED_AT 	| TIMESTAMP  	| 참여 시간 	|

---

↓

↓

집계 출력

종료 또는 재실행

---

## 6. 기대 효과

사용자의 취향 데이터를 수집할 수 있으며, 항목별 응답 현황과 백분율을 통해 전체 응답자의 선호도를 확인할 수 있다.
