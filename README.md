# 개취 설문조사 프로그램

## 프로젝트 소개

사용자의 개인 취향을 조사하고 결과를 데이터베이스(MySQL)에 저장하는 설문조사 프로그램입니다.

사용자는 다음 항목에 대해 응답할 수 있습니다.

* 생년월 (YYYYMM)
* 성별
* 오방색 선호도
* 평면모양 선호도
* 입체모양 선호도

설문 결과는 데이터베이스에 저장되며, 현재까지의 집계 현황과 백분율을 확인할 수 있습니다.

---

## 개발 환경

* Python 3.x
* MySQL
* pymysql

---

## 프로젝트 구조

```text
Project
│
├─ Main_Survey.py
├─ Func_Info.py
├─ Union_DB.py
└─ README.md
```

### Main_Survey.py

프로그램의 시작 파일

기능

* 메뉴 출력
* 설문 참여
* 집계 조회

### Func_Info.py

설문 입력 및 입력값 검증

기능

* 생년월 입력 검증
* 성별 선택
* 오방색 선택
* 평면모양 선택
* 입체모양 선택

### Union_DB.py

데이터베이스 처리

기능

* 테이블 생성
* 설문 결과 저장
* 설문 결과 집계

---

## 데이터베이스 구조

테이블명 : Survey_INFO

| 컬럼명        | 설명          |
| ---------- | ----------- |
| NO         | 자동 증가 번호    |
| BIRTH      | 생년월(YYYYMM) |
| SEX        | 성별          |
| FIVECOLOR  | 오방색         |
| RP         | 평면모양        |
| PS         | 입체모양        |
| CREATED_AT | 설문 참여 시간    |

---

## 실행 방법

1. MySQL 서버 실행
2. 데이터베이스(testdb) 생성
3. 프로그램 실행

```bash
python Main_Survey.py
```

---

## 주요 기능

### 설문 참여

메뉴에서 1번 선택

* 생년월 입력
* 성별 선택
* 오방색 선택
* 평면모양 선택
* 입체모양 선택

입력 완료 후 DB 저장

### 집계 조회

메뉴에서 2번 선택

항목별

* 응답 수
* 백분율

확인 가능
